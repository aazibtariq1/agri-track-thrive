# Agri Track Thrive (formerly AgriManager)

**Current Status**: Work in Progress / Under active development.

## The Problem
Agricultural planning and tracking can be complex, and finding the right information at the right time is difficult. Agri Track Thrive is designed to help local farmers manage their productivity more effectively by providing a centralized platform for tracking crops, yields, expenses, income, and local market prices.

## Tech Stack
We are building this application with **Python** and advanced **AI tools** to empower farmers with actionable insights and smart recommendations. *(The user interface is built using React, TypeScript, and Vite).*

## Running Locally

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/aazibtariq1/agri-track-thrive.git
   ```

2. Navigate to the project directory:
   ```bash
   cd agri-track-thrive
   ```

3. Install the necessary dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```
